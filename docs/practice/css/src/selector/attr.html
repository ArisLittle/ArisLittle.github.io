基本用法 
      
  input[type="submit"] {
    cursor: pointer;
  }
^ 匹配字符开头 
      
  a[href^="https:"] {
    color: green;
  }
  • https://tencent.com
  • http://alibaba.com
  • https://google.com
$ 匹配字符结尾 
      
  img[src$=".jpg"] {
    border: 2px solid orange;
  }
其他
  • a[href*="/home/"]
  • a[rel~=next]
  • a[lang|=en]
