注意，还有一个神奇的现象。如果把box-static的opacity改成0.9（或者其他(0,1)之间的值），static块的z-index会瞬间改变，具有非static类型的效果。在以上的例子中，static由于在最后一个顺位，那么提升之后的z-index，将超过它所有之前非static的。
