历史轨迹

HTML（HyperText Markup Language） 与 CSS（Cascading Style Sheets）

  • 1990年: Tim Berners-Lee 发明 HTML
  • 1994年: Tim Berners-Lee 为了 Web 发展而成立了 W3C
  • 1996年: CSS1 成为 W3C 推荐标准（font, color, margin ...）
  • 1996年: W3C 接管了 HTML 的标准化工作，并在 1 年后发布了 HTML 3.2 推荐标准
  • 1998年: CSS2 成为 W3C 推荐标准（float, position, >, ~, +, * ...）
  • 1999年: HTML 4.0 发布，并在 2000 年成为 ISO (en-US) 标准
  • 2002年: CSS2 Revision 1 在 W3C 发布（修正，清理）
  • 2009年: W3C 完全放弃 XHTML，正式接纳 WHATWG 指定的新标准，就是后来的 HTML5
  • 2011年: CSS2.1 成为 W3C 推荐标准（距离 CSS3 启动已经10多年）

HTML初期的样式与标签混乱

当 Web 变得一团糟之际，CSS 作为解决方案问世了。

CSS 的初衷是把 HTML 混在一起的表现性标记提取出来，使其自成体系，达到结构与表现分离。

结构化、富含语义的 HTML

文档的基础语义应该永远不打折扣。

  • h1, h2
  • p, ul, ol, li, dl, dt
  • strong, em
  • blockquote, cite
  • pre, code, time
  • figure, figcaption
  • fieldset, legend, label
  • thead, tbody, tfoot
  • section, header, footer, nav, article, nav, manu, aside, main
没有真正的 CSS3 规范

CSS3 包含了一系列级别独立的模块。我们提的 CSS3，是指所有足够新的 CSS 规范模块。

  • 如果对之前 CSS 概念的改进，就从 level 3 开始命名

    • CSS Colors Level 3
    • CSS Backgrounds and Borders Level 3
    • CSS Selectors Level 4

  • 如果是全新的模块，从 level 1 命名

    • CSS Grid Layout Level 1
